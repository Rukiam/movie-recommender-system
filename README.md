# movie-recommender-system
Movie recommendation and analytics system using Python and SQL


## 📌 Project Overview
This system analyzes movie data and recommends similar movies
based on user preferences and ratings.

## Tech Stack
- Python (pandas, sqlite3, matplotlib, sklearn)
- SQL (SQLite)
- Jupyter Notebook

## Project Structure
- /data → raw dataset files
- /notebooks → Jupyter analysis notebooks
- /sql → SQL schema and queries
- /images → charts and visualizations

##  Dataset
MovieLens Small Latest Dataset — 9,742 movies, 100,836 ratings
